# Docs Site Readiness

Use when a selected `distribution_surfaces[]` entry has type `docs_site` or when docs are a major launch surface.

## Scope

- Docs URL:
- Hosting provider:
- Build command:
- Deploy command:
- Search provider, if any:

## Checks

- [ ] Docs build succeeds from clean checkout.
- [ ] Quickstart path is complete and copyable.
- [ ] API/reference pages match shipped code.
- [ ] Examples use current package names and versions.
- [ ] Broken links checked.
- [ ] Mobile layout checked.
- [ ] Canonical URL, title, description, favicon, and social image checked.
- [ ] Privacy/terms/support/security links are present if the site collects data or advertises support.
- [ ] Analytics are disclosed or out of scope.
