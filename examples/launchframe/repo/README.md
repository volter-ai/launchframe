# Launchframe

Launchframe creates auditable launch workspaces for OSS developer-tool launches.

```sh
npx launchframe-cli init my-launch
cd my-launch
npx launchframe-cli validate .
```

## What You Get

- Product and org context files.
- HN/Reddit launch drafts.
- Repo readiness and policy/trust templates.
- Visual workflow and website readiness guidance.
- Evidence report and launch-day runbook.
- Pending blocker queue.

## Dogfood Example

This repository includes `examples/launchframe/`, the Launchframe launch created with Launchframe itself.
