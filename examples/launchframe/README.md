# Launchframe Dogfood Example

This directory is the dogfood output for launching Launchframe with Launchframe.

It contains:

- `site/` — the generated marketing site and Startup Pilot capture page.
- `launch-context/` — a concise launch workspace for Launchframe itself.
- `repo/` — generated OSS repo collateral used by the public package/repo.

This is an example of Launchframe output. The reusable product source lives at the repository root.
