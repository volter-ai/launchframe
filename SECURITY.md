# Security Policy

## Reporting A Vulnerability

Please use GitHub Security Advisories for private vulnerability reports:

https://github.com/volter-ai/launchframe/security/advisories

Do not open a public GitHub issue for a suspected vulnerability.

## Scope

In scope:

- The Launchframe CLI.
- The Launchframe templates.
- The public website and example launch workspace in this repository.

Out of scope:

- Third-party services not controlled by Launchframe.
- Social engineering.
- Denial-of-service testing.
